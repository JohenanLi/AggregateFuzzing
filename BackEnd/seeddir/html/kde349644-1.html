The folder permissions part is OK for me, others might disagree.
The "places" bookmarking feature is good. 
What I had in mind was similar to the "network folders" search ability, but on the local computer.

At the moment, if user1 shares a folder or file with user2, they also need to provide the entire path to that folder or file, because otherwise user2 has no choice but to guess the path.

With hindsight what I should have asked for is the ability to recursively change the permissions of the path to the shared folder or file, from the home directory, similar to the option of recursively changing the permissions on any sub folders or files when changing permissions on a folder, but only the path, not the entire tree, and not other contents in the tree.

Right click on folder, select "property’s", in the permissions tab at the bottom, at the bottom under the check box option "Apply changes to all subfolders and their contents", add a new check box to "Share path back to home directory".
This way user2 can see the full path to the shared folder, and only that path and not any other files or folders, and does not need to guess their way to the shared file or folder.
