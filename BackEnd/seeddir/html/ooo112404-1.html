Setup - Bundled Extensions



Specification:
http://wiki.services.openoffice.org/wiki/Setup_-_Bundled_ExtensionsTestcase:

Testcase1:

- start setup.exe (Windows) or setup script (Solaris + Linux)
- walk through the installation dialogs to the „Setup Type“ (Windows) or “Selecte the Type of Installation“ (Solaris + Linux)
- choose the „Custom Installation“ and click next
- in the dialog “Custom Setup” (Windows) or “Select Componets” (Solaris + Linux) expand the “Optional Components” modules
- the module “Dictionaries” is listed under the Java modules [1]
- for en_US there are three dictionaries as submodules: english, french, spanish
- all modules are selected as default
- walk through the installation and install
- start the office
- open ‘Tools – Extension Manager...‘
- all selected dictionaries must be present in Extension Manager

Testcase2:

- start setup.exe (Windows) or setup script (Solaris + Linux)
- walk through the installation dialogs to the „Setup Type“ (Windows) or “Selecte the Type of Installation“ (Solaris + Linux)
- choose the „Custom Installation“ and click next
- in the dialog “Custom Setup” (Windows) or “Select Componets” (Solaris + Linux) expand the “Optional Components” modules
- the module “Dictionaries” is listed under the Java modules [1]
- for en_US there are three dictionaries as submodules: english, french, spanish
- all modules are selected as default
- deselect e.g. spanish
- walk through the installation and install
- start the office
- open ‘Tools – Extension Manager...‘
- all selected dictionaries must be present in Extension Manager
- spanish must not be present in Extension Manager
