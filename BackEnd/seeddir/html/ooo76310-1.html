Status: Preliminary

-


Download and open the testdocument.
Go to the File Menu and select 'Preview in Webbrowser'
After a moment the default browser should start and display the exported temporary HTML file.

Not expected visual discrepancies between Browser and Calc are most likely subject of the HTML filter and should be checked there.
