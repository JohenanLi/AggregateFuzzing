Test purpose:

Test that excessive flickering does not occur

Known issues:

Preconditions of test:

Make sure the OverlayBuffers for Calc and Writer are switched on: Shutdown the office Search in (install)\share\registry\schema\org\openoffice\Office\Common.xcs the flag OverlayBuffer_Writer and OverlayBuffer_CALC and check that they are set to 'true' ATTENTION: Delete the cache-directory (user_profile_dir)/user/registry/cache/ of the office if you had to change the setting!! Pre-rendering Check in the same file as above the flag PaintBuffer_Writer and PaintBuffer_Calc. Must be true. ATTENTION: Delete the cache-directory (user_profile_dir)/user/registry/cache/ of the office if you had to change a setting!!

Test documents:

test.ods

Testcases:

Flickering on scrolling through selected areas

Status: Preliminary

-

  • Download and open the testdoc.ods

  • select the cell range starting at the current cell cursor position up to A1 with the mouse

  • Now use the scrollbar and / or the mouse wheel to scroll the sheet

  • Does the screen flicker with a high rate ? Yes ? File an Issue !

Flickering on moving OLE Objects

Status: Preliminary

-

  • Open a new Calc and a new Writer document

  • Insert some numbers into a cell range

  • Build a Chart out of these data.

  • Deselect the Chart and select it again (green handles)

  • Move the Chart around

  • Does the screen flicker with a high rate ? Yes ? File an Issue !

  • Drag and Drop the Chart on the Writer document. Screen flickering badly ? File an Issue.

  • Insert some Drawing Objects (a Star, a rectangle, a smiley,...)and group these.

  • Move this grouped Object around

  • D&D this Object to the Writer file with pressed CTRL key. The screen flickers ? File an Issue.
