Show formular of regression curve

Status: Preliminary-

  1. Open the spec ChartObjectNames.odt (see references)

  2. Use the first image In chapter 1 for comparison:

    1. File / New / Spreadsheet

    2. Tools / Options / StarOffice / Tabpage general

    3. Mark extended tips for help tips

    4. Leave the Dialog by using ok

    5. Enter in 3 rows and 3 columns random data

    6. Point the cursor into one cell of the just entered data

    7. Insert / Chart

    8. Select xy-Chart as chart type and press Finish

    9. Double click the chart object (inplace mode)

    10. Insert / Statistics...

    11. Choose the Regression Curves Type  linear regression

    12. Leave the Dialog by using ok

    13. Select the regression curve

    14. Check if the formula and r^2 value of regression curve is shown in status bar and quick helpmenu entries have the names described in the specification
