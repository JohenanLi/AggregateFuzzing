Test purpose

Test if annotations persistence in presentation mode works well when "mouse pointer as pen" option is active.

Known issues:

  • no known issues

Preconditions of the test:

  • "Mouse pointer as pen" option active
  • language settings : en-US for the beginning
  • a mouse with right click or a pen of a tablet PC
    • .

Test documents

  • changingpencolorandstrokewidth.odp

Test cases

Annotations persistence during presentation
  • create a new document ODP
  • Add 2 new slides
  • activate "mouse pointer as pen" option in the slideshow settings window
  • start the presentation
  • draw a curve with a given pen color and stroke width (can be changed)
  • Go to Next Slide
  • Come back to Previous Slide
  • Look if annotations reappear with the same color and same stroke width
Annotations persistence with animations in presentation mode
  • load the bugdoc
  • activate "mouse pointer as pen" option in the slideshow settings window
  • start the presentation
  • draw a curve with a given pen color and stroke width (can be changed) and in the upper left corner.
  • Go to Next Slide
  • Come back to Previous Slide
  • Left click to let appear each animation.
  • Look if for each animation, annotations are always on all shapes.
