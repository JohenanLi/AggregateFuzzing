  • Verification Points

    •    The client transmits the message using the following parameters:
    
     IRT   REQ_TIMEOUT 
    
     MRT   REQ_MAX_RT 
    
     MRC   REQ_MAX_RC 
    
     MRD   0 
    
   The client retransmits Request for MRC times.
