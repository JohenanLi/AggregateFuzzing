Test purpose

Test if change pen color and pen stroke width in presentation mode works well when "mouse pointer as pen" option is active.

Known issues:

  • no known issues

Preconditions of the test:

  • "Mouse pointer as pen" option active
  • language settings : en-US for the beginning
  • a mouse with right click or a pen of a tablet PC
    • .

Test documents

  • changingpencolorandstrokewidth.odp

Test cases

Changing pen color during presentation
  • load the bugdoc
  • activate "mouse pointer as pen" option in the slideshow settings window
  • start the presentation
  • draw a curve with default settings for pen color
  • right click
  • click on "change color" in the context menu
  • select a color through the widget which is normally opened.
  • draw a new curve
Changing pen stroke width during presentation
  • load the bugdoc
  • activate "mouse pointer as pen" option in the slideshow settings window
  • start the presentation
  • draw a curve with default settings for pen color
  • right click
  • put the mouse over "pen width" in the context menu
  • select a stroke width through the different items (5).
  • draw a new curve
Testing pen stroke width and color persistence during presentation
  • load the bugdoc
  • activate "mouse pointer as pen" option in the slideshow settings window
  • start the presentation
  • choose a color and a stoke witdh (as described in the first two tests)
  • draw a line
  • go to next slide
  • draw a new line and check if pen color and stoke width have changed.
